# AOC 2021
https://adventofcode.com/2021

## Instructions
- `export PATH=$PATH:$PWD`
- `export AOC_COOKIE=<your session cookie>`
- `virtualenv venv --python=python3.8`
- `pip install -r requirements.txt`
